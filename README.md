# Travel Explorer Website

A simple and responsive **Travel Explorer** website built using **HTML, CSS, and Bootstrap**.  
This project showcases travel destinations with a modern navbar, image carousel, destination cards, contact form, and footer.

## Features

- Responsive navigation bar
- Sticky top navbar
- Bootstrap image carousel
- Carousel indicators and controls
- Destination cards with images
- Card hover animation
- Contact form section
- Footer section
- Mobile-friendly layout
- Clean and attractive travel website design

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5

## Project Structure

```text
travel-explorer/
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

No installation is required.

## Sections Included

### 1. Navbar

The website contains a sticky navigation bar with links for:

- Home
- Gallery
- Contact

### 2. Hero Carousel

The hero section uses a Bootstrap carousel to display travel images such as:

- Mountains
- Beaches
- Cities

### 3. Destination Cards

The destination section includes cards for:

- Mountains
- Beaches
- Cities

Each card contains an image, short description, and Explore button.

### 4. Contact Form

The contact section includes fields for:

- Name
- Email
- Message

### 5. Footer

The footer contains copyright information and a short travel tagline.
